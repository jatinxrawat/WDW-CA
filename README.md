# Student Portfolio Website

This is a simple **student portfolio website** created using **HTML and CSS only**.  
The project is made for academic purposes to practice basic front-end web development concepts.

---

## Project Description

The website介绍 (shows) basic information about a student, including:
- Home section
- About Me section
- Skills & Projects section
- Contact form

All sections are included in a clean layout with simple styling.

---

## Technologies Used

- HTML5
- CSS3

(No JavaScript is used.)

---

## Features

- Simple navigation bar  
- About Me section with profile image  
- Skills/Projects displayed using Flexbox  
- Contact form with basic fields  
- Clean and readable design  

---

## Files Used

- `index.html` – Main HTML file
- `style.css` – CSS file for styling

---

## How to Run

1. Download or clone the project.
2. Open `index.html` in any web browser.
3. The website will load directly without any server.
